### Customer Features

- **Product Catalog**: Browse products across multiple categories (Smart Phones, Laptops, Women Fashion, Kids Fashion, Men Fashion, Skin Care)

- **Brand Management**: Organize products by brands
- **Offers Management**: Create and manage promotional offers
